# MUJ INTURNS – Smart Internship Matching & Skill Readiness Platform

A comprehensive, AI-powered internship platform that bridges the gap between students and companies using intelligent skill matching, verified credentials, and personalized readiness scoring.

## 🚀 Features

### For Students
- **Internship Readiness Score** – Real-time assessment of job readiness (0–100%)
- **AI-Powered Matching** – Intelligent internship recommendations with match scores
- **Personalized Roadmap** – Step-by-step skill development learning path
- **Verified Skills System** – Blockchain-backed credential verification
- **Progress Analytics** – Visual tracking with charts (line, bar, radar)
- **Profile Management** – Professional resume-style profiles with portfolio

### For Companies
- **Internship Posting** – Easy-to-use posting interface
- **AI Candidate Search** – Advanced filtering by skills, college, readiness %
- **Verified Talent Pool** – Access to verified student profiles with match scores
- **Application Management** – Streamlined review process

### Platform
- 🌓 Dark / Light Mode Toggle
- 🪟 Glassmorphism UI with backdrop blur
- 📱 Fully Responsive (Mobile, Tablet, Desktop)
- ✨ Smooth Animations & Micro-interactions
- ♿ Accessible (ARIA labels, keyboard navigation)

## 📱 Pages

| Page | Route | Description |
|------|-------|-------------|
| Landing | `/` | Hero, problem statement, how-it-works, testimonials |
| Login | `/login` | Email/password authentication |
| Signup | `/signup` | Dual role signup (Student / Company) |
| OTP Verify | `/verify-otp` | 6-digit code verification |
| Student Dashboard | `/student/dashboard` | Readiness score, skills, AI recommendations |
| Student Roadmap | `/student/roadmap` | Visual timeline learning path |

| Company Dashboard | `/company/dashboard` | Postings, candidate search, applications |

## 🛠 Tech Stack

- **Framework**: React 18 + TypeScript
- **Routing**: React Router v7
- **Styling**: Tailwind CSS v4
- **Charts**: Recharts
- **Icons**: Lucide React
- **Animations**: Motion (Framer Motion)
- **Build Tool**: Vite

## 🎯 Problem It Solves

1. **Skill Gap** – Students don't know what skills they're missing for their target roles
2. **Fake Credentials** – Companies can't verify student qualifications reliably
3. **Poor Matching** – Generic algorithms don't provide personalized, accurate matches

## 💡 Innovation

- AI-based skill gap analysis
- Blockchain-backed skill verification
- Personalized learning roadmaps
- Real-time readiness scoring
- Intelligent matching algorithm

## 🏃 Running the Project

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:5173/`
