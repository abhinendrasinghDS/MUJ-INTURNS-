Design a modern web application UI for a platform called "MUJ INTURNS – Smart Internship Matching & Skill Readiness Platform".

GOAL:
Create a dual-sided platform connecting students and companies using AI-based skill matching, internship readiness scoring, and course recommendations.

STYLE:
- Modern, minimal, futuristic UI
- Dark + Light mode toggle
- Glassmorphism cards
- Soft shadows, rounded corners (12–16px)
- Clean spacing and grid layout

COLORS:
- Primary: #2563EB (Blue)
- Secondary: #7C3AED (Purple)
- Accent: #10B981 (Green)
- Background: Light (#F9FAFB) / Dark (#0F172A)

FONT:
- Inter or Poppins

-------------------------------------

SCREENS TO DESIGN:

1. LANDING PAGE
- Hero section with heading: "Bridge Your Skills to Real Internships"
- Subheading: AI-based matching and verified skills
- CTA buttons: "Get Started as Student", "Hire Talent"
- Sections:
  - Problem (skill gap, fake resumes)
  - Solution (AI matching, verified skills, readiness score)
  - How it works (3 steps)
  - Testimonials / stats
  - Footer

-------------------------------------
